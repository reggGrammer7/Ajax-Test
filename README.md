AJAX Crash Course – Vanilla JavaScript & PHP
This repository contains a simple practice project demonstrating the core concepts of AJAX using vanilla JavaScript and a lightweight PHP backend. 
It walks through how to send asynchronous requests, load external data, and update the page dynamically without reloading.

What’s Inside
Basic AJAX requests with XMLHttpRequest

Loading text files and JSON data

Displaying and formatting server responses

Submitting forms to a PHP script using AJAX

Multiple example pages (ajax.html, ajax2.html, ajax3.html, ajax4.html)

Sample backend files (process.php, users.json, user.json, sample.txt)

How to Run
Install XAMPP or any local PHP server

Place the project folder inside your server’s htdocs directory

Start Apache (and MySQL if needed)

Open the example files in your browser:

Code
http://localhost/Ajax-Test/ajax.html
Purpose
This repo serves as a hands‑on crash course for beginners learning how AJAX works behind the scenes—no frameworks, no libraries, just pure JavaScript and PHP.
